TiddlyWiki Saver settings
These settings apply to all TiddlyWiki files. Backup files are deleted only during subsequent saves and only when they trigger either of the conditions.
How many recent backup files should be kept. Zero means, no backups. Valid values are from 0 to 100. When empty the default of 3 is used.
How much space in total can the backup files take (in MB). Valid values are from 10 to 1000. Leave this empty to lift if you don't want to set the size limit.
Hide "downloads shelf" from browser interface. This is a global option, not limited to TiddlyWiki files.
